Typographical Conventions

Through this documentation, typefaces are used in the following manner:

  • This font is used for text, with italic used for stack comments, filenames, symbols, and the first appearance of new terms.
  • Headings are shown in a sanserif Bold font.
  • Forth keywords are shown in bold or fixed-font, depending on context.
  • Code snippets and interactive text are shown in a fixed-font.
  • <enclosed> indicates replaceable text in program source.
