This directory contains the following files:

dpans94.pdb: The ISO/ANSI Forth Standard, in PalmPilot DOC format
stackdia.pdb: Quartus Forth stack diagrams for all Palm OS systraps, DOC format
stackdia.txt: The information from stackdia.txt in text format
errlist.pdb: Palm OS error codes, listed alphabetically with values in both
decimal and hex, including any available description. DOC format
errlist.txt: The information from errlist.txt in text format
