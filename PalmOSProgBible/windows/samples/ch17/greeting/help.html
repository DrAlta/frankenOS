Formal Greeting Help

The fields on the home page have the following meanings:

  • Title. Sets the title by which you wish to be addressed in the greeting.
  • First Name and Last Name. Enter your first and last names in these text fields.
  • Time. Enter the time of day for which you wish to be greeted, or leave the time picker alone to use the current time of day according to your handheld's clock.
